# Currency Exchange (Sprint 1)

## Sprint Goal
Пользователь выбирает две валюты, вводит сумму и получает результат конвертации по заданным локальным курсам.

## Features (Sprint 1)
- Выбор валюты "из" и "в"
- Ввод суммы
- Показ курса пары
- Конвертация и округление до 2 знаков
- Swap валют
- Валидация ошибок ввода
- Локальные курсы без API (rates.json)
